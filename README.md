<h1 align="center"> WMS – Sistema Simplificado de Gestão de Armazéns </h1>

<h4 align="center">Projeto acadêmico desenvolvido para a disciplina de Gestão de Projetos – 7º Período.</h4>

## Metodologia

O projeto está sendo desenvolvido utilizando a metodologia ágil Scrum, com divisão em Sprints, reuniões de planejamento, revisão e retrospectiva.

## Tecnologias

- PHP
- Laravel
- SQLite
- Git / GitHub

## Objetivo

Desenvolver um sistema simplificado de gestão de armazéns aplicando na prática conceitos de gerenciamento de projetos ágeis.
